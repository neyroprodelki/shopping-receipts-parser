# 🛒 Парсер чеков (Пятёрочка / Перекрёсток)

Автоматизируем сбор финансовых данных! Этот скрипт создан для того, чтобы за пару минут выгрузить историю ваших покупок из X5 Club за последний год и превратить их в удобный CSV-файл (Excel). 

Вместо ручного переноса трат в таблички — запускаем парсер, получаем чистые данные и отправляем их в любую LLM (ChatGPT, Claude, Gemini) для глубокого анализа бюджета и поиска скрытых закономерностей.

## 🚀 Возможности
* **Глубокая выгрузка:** Собирает детализированные чеки за последние 12 месяцев.
* **Семейный бюджет:** Позволяет добавлять чеки нескольких членов семьи в один сводный файл.
* **Локальная безопасность:** Скрипт работает исключительно на вашем компьютере. Никакие пароли, номера телефонов или финансовые данные не отправляются на сторонние серверы.

## ⚙️ Установка и запуск (для работы с исходным кодом)

Если вы хотите запустить скрипт через Python, вам потребуются установленные библиотеки. 

1. Склонируйте этот репозиторий или просто скачайте файл `auto_parser.py`.
2. Установите необходимые зависимости через терминал:
   ```bash
   pip install requests selenium undetected-chromedriver

```

3. Запустите скрипт:
```bash
python auto_parser.py

```


4. В открывшемся окне браузера авторизуйтесь в X5 Club по номеру телефона (через СМС). Как только вы войдете в профиль, скрипт перехватит нужные токены и начнет сбор данных в консоли.
