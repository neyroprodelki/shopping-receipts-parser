# 🛒 Сборщик чеков (Пятёрочка & Магнит)

Автоматизируем сбор финансовых данных! Этот скрипт создан для того, чтобы за пару минут выгрузить историю ваших личных покупок из сетей **X5 Club (Пятёрочка/Перекрёсток)** и **Магнит** за последний год. На выходе получается единый семейный CSV-файл (Excel).

Вместо ручного переноса трат в таблички — запускаем автоматизацию, получаем чистые данные и отправляем их в любую LLM (ChatGPT, Claude, Gemini) для глубокого анализа бюджета и поиска скрытых закономерностей.

## 🚀 Возможности
* **Две крупнейшие сети:** Бесшовная выгрузка собственных чеков из X5 и Магнита в один файл.
* **Семейный бюджет:** Программа умеет собирать данные с нескольких карт разных членов семьи (при их личном участии и согласии), добавляя в таблицу колонку «Владелец».
* **Продвинутая браузерная автоматизация:** Использует `seleniumbase` для корректного взаимодействия с веб-интерфейсами сайтов и плавной имитации действий реального пользователя (включая автоматическую прокрутку истории покупок).
* **Локальная безопасность:** Скрипт работает исключительно на вашем компьютере. Никакие пароли, номера телефонов или финансовые данные не отправляются на сторонние серверы или разработчику.
