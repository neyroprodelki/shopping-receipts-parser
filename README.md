# 🛒 Парсер чеков (Пятёрочка & Магнит)

Автоматизируем сбор финансовых данных! Этот скрипт создан для того, чтобы за пару минут выгрузить историю ваших покупок из сетей **X5 Club (Пятёрочка/Перекрёсток)** и **Магнит** за последний год. На выходе получается единый семейный CSV-файл (Excel).

Вместо ручного переноса трат в таблички — запускаем парсер, получаем чистые данные и отправляем их в любую LLM (ChatGPT, Claude, Gemini) для глубокого анализа бюджета и поиска скрытых закономерностей.

## 🚀 Возможности
* **Две крупнейшие сети:** Бесшовная выгрузка чеков из X5 и Магнита в один файл.
* **Семейный бюджет:** Программа умеет собирать данные с нескольких карт разных членов семьи, добавляя в таблицу колонку «Владелец».
* **Умный обход защит:** Использует `seleniumbase` для имитации действий реального человека, автоматически прокручивая историю покупок до самых старых чеков.
* **Локальная безопасность:** Скрипт работает исключительно на вашем компьютере. Никакие пароли, номера телефонов или финансовые данные не отправляются на сторонние серверы.

## ⚙️ Установка и запуск (для работы с исходным кодом)

Скрипт содержит функцию умной автоустановки, но для надежности лучше поставить библиотеки вручную:

1. Склонируйте этот репозиторий или просто скачайте файл `auto_parser.py`.
2. Установите необходимые зависимости через терминал:
   ```bash
   pip install requests selenium seleniumbase

```

3. Запустите скрипт:

```bash
python auto_parser.py

```


4. Следуйте подсказкам в консоли. Скрипт спросит, чью карту вы сейчас обрабатываете, откроет браузер и попросит авторизоваться по номеру телефона.
5. **Важно для Магнита:** После ввода СМС просто откиньтесь на спинку кресла. Скрипт сам пролистает вашу историю до конца года и выгрузит детализацию каждого чека!

## 🧠 Как анализировать полученный CSV?

Самый быстрый способ внедрить GTD-подход в финансы — загрузить итоговый файл в нейросеть с таким промптом:

> *"Перед тобой история моих продуктовых покупок за год. Выдели топ-10 самых затратных позиций, разбей траты по категориям и дай рекомендации: какие продукты выгоднее закупать оптом или реже, чтобы оптимизировать бюджет и сэкономить время на походы в магазин?"*
